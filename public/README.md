# Public Folder

This folder contains static assets that are served directly by Next.js.

## Favicon Files Needed

Place your favicon files here:
- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png

Download from: https://jebitech.com/wp-content/uploads/2022/08/JeBI_favicon_dark-300x300.png

Use a favicon generator like https://favicon.io/ to create all required sizes.
